# ✈️ Smart Flight Price Advisor

A machine learning-powered web application that predicts flight prices and recommends the best time to book.

---

## 🚀 Why this project?

Flight prices fluctuate frequently. This system helps users:

* Predict ticket prices
* Decide whether to **Buy Now or Wait**
* Identify optimal travel timing

---

## ⚡ Features

* 💰 Flight price prediction using Random Forest
* 📊 Buy / Wait recommendation system
* 📅 Best day to travel suggestion
* 🌐 FastAPI backend
* ⚡ Lightweight HTML + JavaScript frontend
* 🔌 Real-time API-based architecture

---

## 🧠 Tech Stack

* Python (Pandas, Scikit-learn)
* FastAPI
* HTML, CSS, JavaScript
* Uvicorn

---

## 🖥️ System Architecture

User → Frontend → FastAPI → ML Model → Response
